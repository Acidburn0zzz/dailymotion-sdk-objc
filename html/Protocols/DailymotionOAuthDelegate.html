dailymotionOAuthRequest:createModalDialogWithView:

This delegate method is called when the Dailymotion SDK needs to show a modal dialog window to the user in order to ask for end-user permission to connect your application with his account. You may implement this method only if you choose DailymotionGrantTypeAuthorization grant type. In response to this delegate, your application have to present the given view to the end-user.

- (void)dailymotionOAuthRequest:(DMOAuthClient *)client createModalDialogWithView:(UIView *)view

Parameters

client

The DMOAuthClient sending this message.

view

The view to display to the end user.

Discussion

This delegate method is called when the Dailymotion SDK needs to show a modal dialog window to the user in order to ask for end-user permission to connect your application with his account. You may implement this method only if you choose DailymotionGrantTypeAuthorization grant type. In response to this delegate, your application have to present the given view to the end-user.

The default implementation of this delegate method is to create a new window with the content of the view.

You MUST implement this delegate method if you set the grant type to DailymotionGrantTypeAuthorization.

dailymotionOAuthRequest:didRequestUserCredentialsWithHandler:

Called when the grant method is DailymotionGrantTypePassword and the credentials are requested by the library. The delegate have to ask the user for her credentials and need to call back the handler method in order for the pending API calls to be completed.

Parameters

Discussion

Called when the grant method is DailymotionGrantTypePassword and the credentials are requested by the library. The delegate have to ask the user for her credentials and need to call back the handler method in order for the pending API calls to be completed.

You MUST implement this delegate method if you set the grant type to DailymotionGrantTypePassword.

dailymotionOAuthRequest:shouldOpenURLInExternalBrowser:

When a link is clicked in the authorization dialog, like for instance to create an account or recover a lost password, the Dailymotion SDK asks the UI delegate if the link should be openned in an external browser or if the UI delegate want to handle the openned link by itself.
