DMPlayerViewController Class Reference

Inherits from UIViewController
Conforms to UIWebViewDelegate
Declared in DMPlayerViewController.h

Overview

The DMPlayerViewController class implements a wrapper around the Dailymtion HTML5 player so it can be easily controlled.
