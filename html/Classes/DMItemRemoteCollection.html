itemsWithFields:forPage:withPageSize:do:

Retrieve items with specified pre-cached fields on the current collection with given pagination information.

- (DMItemOperation *)itemsWithFields:(NSArray *)fields forPage:(NSUInteger)page withPageSize:(NSUInteger)itemsPerPage do:(void ( ^ ) ( NSArray *items , BOOL more , NSInteger total , BOOL stalled , NSError *error ))callback

Parameters

fields

The list of fields to retrieve for each items of the list.

page

The page offset.

itemsPerPage

The number of items per page to retrieve.

callback

The block to be called with the result.

Discussion

Retrieve items with specified pre-cached fields on the current collection with given pagination information.

The data may come from cache or network. If cached data are stalled, the block will be called twice. First time the data will come from the stalled cache, the stalled parameter is then set to YES. In parallele, an API request is automatically performed to retrieve fresh data. On success the block is called a second time with the stalled parameter set to NO.
